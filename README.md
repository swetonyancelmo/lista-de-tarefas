# 📝 Lista de Tarefas

**Projeto simples de lista de tarefas**, criado com **HTML, Tailwind CSS e JavaScript puro**. Ideal para aprendizado e demonstração de boas práticas de front-end.

---

## 🎯 Funcionalidades

- ✅ Adicionar tarefas dinamicamente
- ✍️ Marcar tarefas como concluídas
- ❌ Remover tarefas individualmente
- 💾 Persistência via `localStorage` (opcional)
- 📱 Responsivo e estiloso com Tailwind

---

## 🛠 Tecnologias e Ferramentas

- **HTML5** – estrutura semântica
- **Tailwind CSS** – estilização rápida e responsiva
- **JavaScript (ES6+)** – DOM, eventos
- **localStorage** – Salva as tarefas no navegador

---

## 📚 Aprendizados e desafios

- Manipulação dinâmica do DOM (createElement, appendChild, etc.)
- Eventos (click, keydown, focus, etc.)
- Gestão de estado simples com localStorage
- Estilização responsiva com Tailwind e configuração de estados (hover, focus)

## ✅ Boas práticas

- Sem frameworks pesados: foco em JavaScript puro
- Código modular e limpo
- Seletor de foco personalizado (focus:outline-none, focus:border-*)
- Design responsivo pronto para ser expandido
